# Editor setup

On IBM-managed machines, install
[IBM Bob](https://bob.ibm.com/docs/ide/getting-started/install) through the approved route.

If Bob or another VS Code-based editor offers them, install:

- **Tailwind CSS IntelliSense** (`bradlc.vscode-tailwindcss`) for class completion and previews
- **Biome** (`biomejs.biome`) for the repository's formatter and linter

Ask before installing extensions. TypeScript support is normally built in; Docker, Git, and
large extension packs are not required.
